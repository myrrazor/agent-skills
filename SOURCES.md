## Independent research

- Shapkin et al., **RuBench: A Repository-Level Agentic Coding Benchmark with Natively Authored Russian Task Specifications**, arXiv:2607.06411. The paper reports safeguard fallback on 5 of 25 Fable/Claude Code tasks in its single hors-concours run and documents mid-session model substitution.  
  https://arxiv.org/html/2607.06411

## Interpretation note

Fable 5 is new, and public independent guardrail research remains sparse. Most precise behavioral documentation currently comes from Anthropic. Public issue reports can reveal useful failure modes but are anecdotal and should not be treated as calibrated trigger probabilities.
